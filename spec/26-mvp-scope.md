## §26. MVP Scope

## §26.1 MVP Must Have

```text
manual/assisted plan import
concept files
material files
material parts
suggested routes
state YAML
trajectory template
influence field YAML
graph builder
graph JSON
static viewer
boundary checker
one imported example plan
```

## §26.2 MVP Can Skip

```text
full automatic material scraping
semantic embeddings
large interactive UI
multi-user support
cloud sync
database
auth
complex analytics
automatic confidence upgrades
```
